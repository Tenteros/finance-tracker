# 💰 Advanced Finance Tracker

🇺🇸 **[English](#english) | 🇷🇺 [Русский](#русский)**

---

## 🇺🇸 English <a name="english"></a>

A modern, responsive personal finance tracker built with Vanilla JS, Vite, and Supabase. Features secure authentication, an interactive dashboard, custom categories, i18n support, and dark/light modes.

🚀 **Live Demo:** [advanced-finance-tracker-seven.vercel.app](https://advanced-finance-tracker-seven.vercel.app/)

### 🛠️ Tech Stack
* **Frontend:** Vanilla JS (ES6+), HTML5, CSS3
* **Build Tool:** Vite
* **Backend/DB:** Supabase (PostgreSQL, Auth, RPC)

### 💻 Local Setup

1. Clone the repository and install dependencies:
```bash
git clone https://github.com/Tenteros/finance-tracker.git
cd finance-tracker
npm install
```

2. Create a `.env` file in the root directory based on the example:
```env
VITE_SUPABASE_URL=https://xxxxx.supabase.co
VITE_SUPABASE_ANON_KEY=your-anon-key
```

3. Start the development server:
```bash
npm run dev
```
